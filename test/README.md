# Goals of writing tests
- Boosted Confidence
- Quality Code
- Better Documentation
  
  # Identifying what to test
  - Input
    - Data
    - Props
    - User Interaction (e.g. clicks)
    - Lifecycle Methods
    - Vuex
    - Route Params
  - Output
    - DOM Render
    - External function calls
    - Events emitted
    - Route Changes
    - Vuex Store updates
    - Children Connection
